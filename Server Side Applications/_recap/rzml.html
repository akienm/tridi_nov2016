rzml

Reference Sheet

version 1.4.6

Description

rzml is a XML based language used to exchange data between REALVIZ applications.

Root Element

<RZML>

  1. any number of camera frame <CFRM>

<CCSTR> : camera constraint

Description :

defines a constraint on camera parameters. The constraint may be restricted to some specified time ranges.

Attributes :

XML Name Type Mode Default Description
n name string implied   relation name
k kind choice default focal parameter to be shared by several images of the shot
is info status choice implied unknown level of the information
fovx view field real implied   known or estimated field of view (horizontal angle in degrees)
a axis choice implied   axis of the dolly relation or normal axis of the plane relation
ef enabled flag bool implied 1 allows to desactivate the constraint
ux ux real implied 0 uncertainty in abscissa of position constraint
uy uy real implied 0 uncertainty in ordinate of position constraint
uz uz real implied 0 uncertainty in z coordinate of position constraint
camid camid unsigned integer implied 0 id of constrainted camera

Content :

a sequence made of :
  1. an optional position <P>
  2. any number of time range <TRNG>

Types :

camera constraint - kind
XML Name Description
f focal the focal is shared
pan pan the optical center is shared
pln plane 2 coordinates values of the optical center are shared
d dolly 1 coordinate value of the optical center is shared
fix fixed the camera is fixed
pos pos the position is constrainted
camera constraint - info status
XML Name Description
u unknown the shared value is unknown
e estimated the shared value is approximated
k known the shared value is fixed
camera constraint - axis
XML Name Description
x x dolly along the x axis or yz planar constraint
y y dolly along the y axis or xz planar constraint
z z dolly along the z axis or xy planar constraint

<CFRM> : camera frame

Description :

camera parameters at a specific time frame

Attributes :

XML Name Type Mode Default Description
n name string implied   camera name
type camera lens type choice implied Rectilinear the type of camera/lens
t time unsigned integer implied 0 time frame position
cf calibration flag bool implied 0 calibration flag
fovx view field real required   field of view (horizontal angle in degrees)
rd distortion real implied 0 radial distortion parameter number 1 or default one
rd2 distortion2 real implied 0 radial distortion parameter number 2 or default one
rd3 distortion3 real implied 0 radial distortion parameter number 3 or default one
hd high distortion real implied -1 non polynomial radial distortion used to pre-process the images
pr pixel ratio real implied 1 pixel aspect ratio
ppx principal point x real implied 0.5 left based abscissa of the principal point in normalized unit
ppy principal point y real implied 0.5 top based ordinate of the principal point in normalized unit
radius radius real implied 0 radius of the visible circle of a fisheye lens
distoType disto type choice implied disto3d distortion type, define number of parameters and distortion type

Content :

a sequence made of :
  1. one translation <T>
  2. one rotation <R>
